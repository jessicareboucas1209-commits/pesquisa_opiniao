# Pesquisa de Satisfação - Atendimento ao Cliente

Este projeto consiste em um programa desenvolvido em **Python** para coletar e contabilizar o feedback de clientes sobre o atendimento prestado.

## 🚀 Funcionalidades
- Coleta de dados: Nome, Idade e Opinião.
- Classificação de opiniões: 1 (Excelente), 2 (Bom) e 3 (Ruim).
- Uso de estrutura de repetição `FOR` para múltiplos entrevistados.
- Contador automático de respostas positivas (Excelente) e negativas (Ruim).

## 🛠️ Tecnologias Utilizadas
- Python 3.x
- Estruturas de Decisão (`if`, `elif`)
- Estruturas de Repetição (`for`)
